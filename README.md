# NOVA HUB ; Game Center

A futuristic all-in-one browser arcade packed into a single HTML file.  
Built with pure HTML, CSS, and JavaScript — no frameworks, no installs, no dependencies.

NOVA HUB is designed as a stylish mini-game collection featuring fast-paced reflex challenges, math-based brain games, arcade classics, and rhythm experiences inside a neon sci-fi interface.

---

## ✨ Features

- **16 Fully Playable Games**
- Smooth animated futuristic UI
- Dynamic starfield background
- Responsive mobile-friendly design
- Multiple game genres
- No backend required
- Runs entirely in-browser
- Single-file project structure
- Custom neon visual system
- Instant game modal system
- Timers, streaks, combos, scores & progression

## ⚡ CHAOS

### Included Games

  - Hit moles before they disappear.

# 🧠 Technical Highlights

## Modular Game System
Each game is separated into:
- UI generation
- Initialization
- Logic handlers
- State management

## Dynamic Modal Engine
Games open instantly inside reusable modal windows.

## Unified Timer System
Custom interval/timeout cleanup prevents overlapping game states and memory leaks.

## Responsive Layout
CSS Grid + Flexbox adaptive system for mobile and desktop.

- Glow animations
- Animated starfield
